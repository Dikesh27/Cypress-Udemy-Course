# Cypress-Udemy-Course
How to INSTALL CYPRESS
- create cypress folder
- install vs code
- install node js
- add node in node in system environment
	- open "edit the system environment variable"
	- click on "environment variable"
	- click on New System variable
	- add Variable name - NODE_HOME
	- add Variable value - *path on node in pc*
- open node.js command promp
- open cypress folder location
- write "npm -i init" to install node inside the folder
- open folder in vs code
- open git terminal
- write npm install cypress --save-dev to INSTALL CYPRESS
